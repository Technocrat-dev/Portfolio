# Portfolio

Personal portfolio showcasing distributed systems, AI/ML, and full-stack projects.

## 🚀 Tech Stack

- **Framework**: Astro v5
- **UI**: Preact
- **Styling**: TailwindCSS v4
- **Icons**: astro-icon
- **Analytics**: Vercel Speed Insights

## 📦 Installation

```bash
npm install
```

## 🛠️ Development

```bash
npm run dev
```

## 🏗️ Build

```bash
npm run build
```
